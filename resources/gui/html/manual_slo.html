Navodila za uporabo

1. Izbira vhodne datoteke

  • PDF Poročilo: Naložite izvoz vašega posrednika (trnutno le Trade Republic). Program bo samodejno pridobil podatke o trgovanju.
  • JSON Datoteka: Naložite predhodno ustvarjeno datoteko transactions.json, da preskočite fazo branja PDF dokumenta.

2. Načini generiranja

Privzeto program ustvari XML datoteko za eDavke. Vendar pa:

  • Način JSON: Če je ta možnost označena, program le shrani pridobljene podatke v JSON datoteko. To uporabite za preverjanje natančnosti podatkov pred končno izdelavo XML datoteke.

Opomba: Prepričajte se, da sta davčna številka in leto pravilna za glavo XML datoteke.
