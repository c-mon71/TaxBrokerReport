User Manual

1. Input File Selection

  • PDF Report: Upload your broker's export (currently only Trade Republic). The parser extracts trade data.
  • JSON File: Load a previously generated transactions.json to skip the PDF parsing stage.

2. Generation Modes

By default, the app generates an XML for eDavki. However:

  • JSON Mode: If checked, the app only saves the extracted data to a JSON file. Use this to verify accuracy before final XML creation.

Note: Ensure your Tax Number and Year are correct for the XML header.
