## ü™ü Windows

**Requirement:** Windows 10 or 11 (64-bit).

* **Download:** `TaxBrokerReport-Win64.zip`
* **Instructions:** Extract the `.zip` file and run `TaxBrokerReport.exe`.
* **Note:** Because this is independent open-source software, Windows might show a "Windows Protected your PC" popup. Click **"More info"** and then **"Run anyway"**.

## üçé macOS

**Requirement:** macOS 12 (Monterey) or newer.

1. **Check your Chip:** Click the Apple icon (Ô£ø) > "About This Mac".
2. **If it says "Intel":** Download `TaxBrokerReport-macOS-Intel.zip`
3. **If it says "Apple M1/M2/M3/M4":** Download `TaxBrokerReport-macOS-ARM64.zip`

### ‚ö†Ô∏è Important: Bypassing Apple Gatekeeper

Because this is an independent open-source project and does not have a $100/year Apple developer license for official code signing, macOS will initially block it.

**How to allow it (Required only once):**

* Try to open the app. You will see a warning: *"TaxBrokerReport cannot be opened because it is from an unidentified developer."*
* Click **Cancel**.
* Go to **System Settings** -> **Privacy & Security**.
* Scroll down to the **Security** section.
* You will see: *"TaxBrokerReport was blocked from use because it is not from an identified developer."*
* Click **"Open Anyway"** and enter your administrator password.
* On the final popup, click **Open**.
